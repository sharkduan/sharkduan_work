# First-Pass Reaction Family Vocabulary

The first model will support six residue-reaction families: CYS Michael addition, CYS nucleophilic substitution, CYS disulfide exchange, SER acylation, SER phosphonylation, and LYS Schiff base. This intentionally excludes long-tail residue chemistry, multi-residue mechanisms, cofactor covalent modification, and metal coordination so the initial model can learn common covalent attachment patterns with enough data density and tractable validity rules.
