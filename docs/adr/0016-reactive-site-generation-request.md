# Reactive-Site Generation Request

Inference will require a protein structure path, target chain, target residue sequence id, target residue name, target atom name, residue-reaction family, and sample count. Optional controls include pocket radius, generated molecule size range, guidance strength, sampling steps, and random seed. Reference ligands, scaffolds, warhead motifs, and ligand attachment atoms are not required because the task is de novo covalent inhibitor generation with dynamic ligand-side attachment prediction.
