# Data-Derived Rules With Manual Curation

Reaction-family rules will be initialized from the covalent training corpus and manually corrected before use. Manual curation is required for family mapping, target reactive atoms, ligand attachment atoms, allowed and forbidden SMARTS, geometry ranges, valence changes, warhead naming, and abnormal structure exclusion. Purely manual rules are likely too narrow, while unreviewed data-derived rules would preserve noisy database annotations.
