# CovalentInDB 2.0 as a Required Source

CovalentInDB 2.0 is a required source for the covalent training corpus, not an optional reference. The ETL should start with CovBinderInPDB because it is closest to atom-level linkage supervision, but CovalentInDB 2.0 must be integrated to provide broad covalent inhibitor coverage, warhead types, cocrystal structures, covalent binding sites, and reaction mechanism annotations.
