# mmCIF-First Covalent Complex Output

Generated covalent protein-ligand complexes will be saved primarily as mmCIF with structured atom-level covalent linkage records, and optionally exported as PDB with LINK/CONECT records for visualization and older tool compatibility. PDB-only output is rejected because mmCIF better preserves chain, residue, atom, connection type, and distance information for covalent protein-ligand attachments.
