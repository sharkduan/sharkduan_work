# CovalentGenerationResult

Each generated sample will be saved as a CovalentGenerationResult containing the ligand SDF, covalent protein-ligand complex file, predicted ligand attachment atom, predicted covalent edge, reaction family, matched or predicted warhead type, covalent edge score, geometry metrics, validity gate status, failure reason, molecular quality metrics, and covalent docking score. Ligand-only SDF output is insufficient because it loses the protein-ligand covalent attachment context.
